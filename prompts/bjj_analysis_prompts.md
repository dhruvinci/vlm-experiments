# BJJ Analysis Prompts

## General Analysis

```
Analyze this Brazilian Jiu-Jitsu match video. Describe the key techniques, positions, and strategies used by both competitors.
```

## Technique Identification

```
Identify all BJJ techniques used in this video. For each technique, note the timestamp (approximate), the practitioner who applied it, and rate its execution quality on a scale of 1-10.
```

## Position Analysis

```
Analyze the positional control demonstrated in this BJJ match. Which competitor maintained better position throughout the match? What were the key transitions and how did they affect the match outcome?
```

## Strategy Assessment

```
Evaluate the strategic approach of both competitors in this BJJ match. What game plans did they appear to follow? How did they adapt to each other's techniques and positions?
```

## Improvement Recommendations

```
Based on this BJJ match video, provide specific recommendations for how each competitor could improve their performance. Focus on technical execution, position maintenance, and strategic decision-making.
```
