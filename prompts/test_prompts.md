# Test Prompts

## Test 0: Connectivity Test

```
Hi, how can you help me?
```

## Test 1: Image Test

```
Explain with as much detail about what is happening here
```

## Test 2: 3-min Video Test

```
Analyze this Brazilian Jiu-Jitsu match video. Describe the key techniques, positions, and strategies used by both competitors. Give me a sequence of moves that you recognize through this video.
```

## Test 3: 16-min Video Test

```
Break down this video in the following ways:
General Analysis: Analyze this Brazilian Jiu-Jitsu match video. Describe the key techniques, positions, and strategies used by both competitors.
Technique Identification: Identify all BJJ techniques used in this video and provide a sequence. For each technique, note the timestamp (approximate), the practitioner who applied it, and rate its execution quality on a scale of 1-10.
Position Analysis: Analyze the positional control demonstrated in this BJJ match. Which competitor maintained better position throughout the match? What were the key transitions and how did they affect the match outcome? Give a scramble rating vs time graph for the entire match, where scramble rating is how much the players are scrambling from a scale of 1-10 at that moment.
Strategy Assessment: Evaluate the strategic approach of both competitors in this BJJ match. What game plans did they appear to follow? How did they adapt to each other's techniques and positions?
Improvement Recommendations: Based on this BJJ match video, provide specific recommendations for how each competitor could improve their performance. Focus on technical execution, position maintenance, and strategic decision-making.
```
